# c-server

## Brief

This project is implemented in C, based on what I learned from two C++ projects: `30dayMakeCppServer` and `30daysCppWebServer`.

## Usage

```console
$ gcc -o build build.c
$ ./build
$ ./test/server &
$ ./test/client
```
